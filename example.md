# Default slide

A standard content slide on the HCCS green background.

- White text on HCCS green
- `inline code` for terms
- [Links](https://hudcostreets.org) in accent yellow
- Per-slide QR auto-injected in the bottom-left
- Logo + URL footer in the bottom-right

> Blockquotes get the accent-yellow rule on the left.

# Light variant

Set `class: light` on a slide for white-bg / dark-fg.

- Same fonts and rhythm
- Logo flips to the green variant automatically
- Useful for image-heavy or screenshot slides
